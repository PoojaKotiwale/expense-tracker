# Expense Tracker Web Application

A full-stack web app to manage and track daily expenses.

## Tech Stack
PHP | MySQL | Bootstrap | jQuery | JavaScript | AJAX | Chart.js

## Features
- User authentication and registration
- Add, edit, delete expenses
- Category management
- Real-time CRUD using AJAX (no page reload)
- Data visualization with Chart.js
- Responsive Bootstrap UI

## How to Run
1. Clone this repo
2. Import SQL file into MySQL via phpMyAdmin
3. Update includes/db.php with database credentials
4. Run on XAMPP localhost
